Бейдж типа NAT
SNAT → Трансляция источника — исходящий трафик
DNAT ← Трансляция назначения — входящий трафик
SNAT+DNAT ↔ Двойная трансляция источника и назначения
Ассоциированные правила безопасности
ALLOW Правило разрешает трафик
DENY Правило запрещает трафик
#43 pre Позиция в наборе и приоритет (pre / post / default)
Индикаторы покрытия правила безопасности
✓ full Правило безопасности полностью охватывает условия NAT правила по всем трём измерениям (src / dst / svc) — трафик гарантированно разрешён или запрещён
partial Пересечение есть, но правило безопасности не охватывает NAT правило целиком — часть трафика может обрабатываться другими правилами
⚠ src Правило безопасности уже NAT правила по указанному измерению (src / dst / svc) — часть NAT-трафика может быть заблокирована. Например: NAT разрешает все источники, а правило безопасности ограничивает конкретными адресами
Нет ассоциированных правил Для данного NAT правила не найдено ни одного совпадающего правила безопасности — трафик будет заблокирован политикой по умолчанию
